\## 1. Material Specifications



The following units define the standards for all materials used in campus construction.



\### 1.1 Brick Units



1\.  \*\*Brick:\*\* Acme UCA Blend: 240/260 Blend, 60% 240 and 40% 260

2\.  \*\*Texture:\*\* Buff

3\.  \*\*Color:\*\* Shall be selected from approved samples matching existing campus blend.

4\.  \*\*Type:\*\* High-fired clay face brick, Grade SW (ASTM C216).



\### 1.2 Mortar and Grout



\* \*\*Exterior Walls:\*\* Type N Mortar is required for all non-loadbearing exterior applications.

\* \*\*Interior Walls:\*\* Type S Mortar is required for all loadbearing walls and foundations.



\## 2. Quality Assurance Standards



\### Required Testing



| Test Description | Frequency | Required Passing Score |

| :--- | :--- | :--- |

| \*\*Compressive Strength\*\* | Per 5,000 units | Minimum 1900 psi |

| \*\*Absorption Rate\*\* | Per 5,000 units | Maximum 6% |



> \*\*Note:\*\* All quality control testing must be performed by an independent, NCDOT-certified testing agency.
